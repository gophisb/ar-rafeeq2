# الرفيق — تطبيق إسلامي شامل 🌙

**الإصدار 2.0** — نقطة تحوّل معمارية حقيقية: الصفحة الرئيسية الآن تخزّن تقويم الشهر الهجري كاملاً محلياً، فتعمل بدقة رسمية كاملة حتى دون أي اتصال بالإنترنت (لا مجرد تقريب فلكي محلي).

**تطبيق ويب تفاعلي احترافي يجمع بين أدوات إسلامية أساسية في واجهة واحدة متكاملة وآمنة.**

---

## ✨ الميزات الرئيسية

### 📅 **مواقيت الصلاة**
- تحديد دقيق لمواقيت الصلاة الخمس عبر Aladhan API
- التقويم الهجري المتزامن
- اختيار من 69 ولاية جزائرية (التقسيم الإداري المحدَّث 2026، شامل الولايات الـ11 المستحدثة)
- أذان صوتي تلقائي لكل الصلوات الخمس + دعاء تلقائي بعده
- تذكير يوم الجمعة، تذكير الثلث الأخير من الليل، وضع رمضان تلقائي (هلال بدل النجمة + أوقات الإفطار والسحور)
- عدّاد تنازلي حي للصلاة القادمة

### 🕌 **تحديد القبلة**
- بوصلة تفاعلية تحسب الزاوية الجغرافية الدقيقة نحو الكعبة
- تحديد موقع تلقائي عبر GPS
- إبرة حيّة تتبع اتجاه الجهاز (DeviceOrientation API)

### 📖 **القرآن الكريم الكامل**
- 114 سورة، 6236 آية — **رواية حفص عن عاصم** (نص Tanzil الموثق)، **مخزّنة محلياً بالكامل** في `quran-local.json` — تعمل فوراً دون اتصال بالإنترنت من أول استخدام
- **التفسير الميسّر** (إصدار مجمع الملك فهد) يُجلب من Al Quran Cloud API عند فتح كل سورة (يحتاج اتصالاً أول مرة لكل سورة، ثم يُخزَّن دائماً بعدها)
- بحث فوري في أسماء السور عبر فهرس Lunr.js (يدعم أخطاء إملائية بسيطة)

### 📚 **الأربعون النووية**
- **42 حديثاً كاملة** (المتن والشرح الأصيل للإمام النووي رحمه الله، ت 676هـ) — نص تراثي كلاسيكي، لا حقوق ملكية معاصرة عليه
- بحث فوري في نص الحديث والشرح عبر فهرس Lunr.js
- تخزَّن محلياً في `arbaeen-data.json` وتعمل دون اتصال بالإنترنت

### 🌅 **الأذكار**
- 10 أذكار صباحية، 9 أذكار مسائية
- عدادات تفاعلية تُغلق تلقائياً عند بلوغ العدد المطلوب
- شريط تقدّم إجمالي

### 🤲 **حصن المسلم**
- 30 دعاء يومياً لمناسبات الحياة المختلفة
- بحث سريع وإمكانية "تفضيل" الأدعية

## 🛠 التكنولوجيا المستخدمة

- **HTML5 / CSS3 / JavaScript ES6+** — منطق كل صفحة مستقل، والأنماط المشتركة (الألوان، القواعد الأساسية) موحَّدة في `style.css`
- **فحص آلي (CI)** — عبر GitHub Actions (`.github/workflows/check.yml`)، يتحقق تلقائياً عند كل رفع أن كل الروابط الداخلية سليمة وأن ملفات القرآن والأربعين كاملة، قبل وصول أي خطأ للمستخدمين
- **Service Worker** — تخزين مؤقت لأصداف التطبيق وطلبات الـ API (عمل جزئي دون اتصال بعد أول استخدام)
- **PWA** — قابل للتثبيت كتطبيق على الهاتف (manifest مضمّن داخل index.html)
- **Geolocation API + DeviceOrientation API** — لصفحة القبلة
- **Al Quran Cloud API** (alquran.cloud) — نص القرآن والتفسير الميسّر
- **Aladhan API** — مواقيت الصلاة
